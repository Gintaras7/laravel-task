# Test task

## Stack

- PHP 8.4  
- Laravel 12  
- Vue 3  
- Node.js 24.10.0  

## Laravel installation

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate

````


## Vue installation
```bash
npm install # install dependencies
npm run dev # run development server
```

## Running Tests

Run the backend tests with:

```bash
php artisan test
```
